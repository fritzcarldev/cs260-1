Welcome to the cs260 Debugging Tutorial

In order to follow the instructions given in this tutorial you should be using Google Chrome as your web browser. This demo page you are currently viewing represents an index.html and main.js file. When the HTML file is loaded by the browser, it loads the JavaScript file. With the JavaScript loaded you can open it up and debug it using the following steps.

  1. Press F12 to open the Chrome debugger.
  2. Switch to the Sources Tab.
  3. Use the Open file command (⌘-P on MacOS).
  4. Select main.js. This should open the file in the sources tab of the debugger.
  5. Put a break point on line 3 by click on the left side of the sourcd pane.
  6. Open the Console pane of the debugger if it is not already by pressing the ESC key.
  7. In the Console pane you can execute any valid JavaScript. For example, type 1+1 this should evaluate to 2. Now call the function displayed in the source pane by typing debugMe() and pressing the Return key.
  8. The debugger should stop on your breakpoint. You can press F9 to step through each line of the function.
  9. Press F8 to continue execution.
  10. Press the gear menu in the upper right corner of the debugger and select Shortcuts to see all the available commands for debugging.
